##VINSSOFT 
Aplicación web para la gestión de eventos y personal del departamento de Vinculación Social de la Universidad Politécnica Territorial de Aragua "Federico Brito Figueroa" 

##Alcances 
Portal para el acceso público a los detalles de los eventos planificados Gestión de solicitudes de uso del salón de usos múltiples Gestión de Eventos Gestión de Personal (Jefe, Coordinadores, Becarios) 
Control de Asistencia Gestión de Horarios Gestión de Tareas de los Becarios (Protocolo, Estudiantina, Coral, Danza) 
Reportes PDF Planificación (Horario+Tareas) 
Eventos Asignados (Futuros o Pasados) 

##Detalles Técnicos 
Diseño Front-End: Bootstrap jQuery Font Awesome 
Desarrollo Back-End: Symfony MySQL
